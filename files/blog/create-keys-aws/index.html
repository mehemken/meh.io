Create an AWS KeyPair with the CLI

15 June 2017

Comment on twitter!

Key Pairs in the CLI

The source of this post is the AWS docs. The reason you need the flags here is that you can create the key pair in the CLI, but if you don't download the key material, you won't have the key on your local machine.

aws ec2 create-key-pair \
    --key-name KeyFooBar \
    --query 'KeyMaterial' \
    --output text \
    > KeyFooBar.pem
chmod 400 KeyFooBar.pem

And that's it folks!
