My intro

Why am I trying to change careers? Because I am fascinated by programming and setting up servers to deliver code. I sit down to write code every day because I love doing it.

I wrote my first computer program when I was 11. Then I became a musician, and athlete and studied Sociology. A year and a half ago I was unhappy with the long term future of my job and I decided to make a change toward computer programming. Today I'm very well versed in Python and I've used many of the buzz word technologies such as docker, mongodb, AWS, Google Cloud, Jenkins, Nginx etc.

Now that I have a year under my belt I've realized that I really like programming. I love working with Python and dabbling in other languages like Go, JavaScript, Rust, C etc. But that's not enough for me. I want to deploy my projects while retaining full control of that deployment.
